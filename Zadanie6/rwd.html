Responsive Web Design
What it means to make a responsive website?

Responsive site is the type of site, which looks good both on small Iphone SE and big TV. It changes size of its elements, so it can looks good.

How to make a responsive website?

First set your viewport in your head on website index. Next use @media and orientation: portrait or landscape in CSS. After that you must to set breakpoint of native resolution. The best that points is 320px, 576px, 768px, 1024px and 1200px. They values are not only which is going to looks good. You can use your own values too.

Why we are using RWD and making responsive websites?

Becouse they can look good on every device. If we are not using RWD, the website will look nice only on the specific resolution. While using RWD we can make website which looks good on every device, Iphone SE and 75" TV.
